Les informations sont amenées à évoluer avec le temps.

Cette base de données a été créée dans le cadre d'une thèse vétérinaire.
Informations complémentaires :

Cette base de données est destinées aux praticiens vétérinaires pour les aider dans leur pratique en rassemblant des informations sur les plantes les plus couramment utiliser en France.

De nombreuses plantes médicinales n'ont pas été incluses dans ce travail mais pourront être ajoutées par la suite si nécessaire.

Enfin, des propositions de traitements pourront être trouvées pour certains grands syndromes. Ces propositions ne sont que ça et n'ont pas vocation à forcer le praticien à les utiliser : elles sont à modifier en prenant en compte le patient.
